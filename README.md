This is a Power BI dashboard built on a fashion boutique Excel dataset.  
The goal is to analyse how discounts, returns and product performance affect total revenue.

## Tools

- Power BI Desktop  
- Power Query (for data cleaning)  
- DAX (for calculations)  
- Excel (fashion_boutique_dataset.xlsx)

## What I did

- Cleaned the Excel data in **Power Query**  
  - Handled null values  
  - Fixed data types  
- Created **DAX measures** such as Total Revenue, Revenue Before/After Discount, Total Quantity, Return %, Average Discount and Average Rating.
- Designed visuals to show:
  - Total revenue and key KPIs  
  - Revenue by brand, category, season and year  
  - Return rate by brand and category  
  - Stock quantity by size  

## How to View

1. Open the `.pbix` file in **Power BI Desktop**.  
2. Ensure the Excel file is available or update the data source path.  
3. Click **Refresh** and interact with the slicers and charts.
